Kodable by Kodnext

What is Kodable?

The name Kodable is derived from two words "code" and "editable".

Kodable is a High level transpiler which can give you an ability to write custom syntexes in any higher level language.

Why we need Kodable?

  • Kodable can ease the life of a programmer. If programmers get choice of writing their own syntexes.
  • Kodable can provide same syntex for various programming languages.

Things to note!

  • FLEX (fast lexical analyzer generator) is a tool/computer program for generating lexical analyzers (scanners or lexers) written by Vern Paxson in C around 1987. flex article jeeks for jeeks.
  • PLY is yet another implementation of lex and yacc for Python. Some notable features include the fact that its implemented entirely in Python and it uses LALR(1) parsing which is efficient and well suited for larger grammars. link to pip package.
  • ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, or translating structured text or binary files. It's widely used to build languages, tools, and frameworks. From a grammar, ANTLR generates a parser that can build and walk parse trees. antlr website.
